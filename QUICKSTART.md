# 🚀 Quick Start Guide

Get your Next.js Extngo site running in 3 steps:

## Step 1: Install Dependencies

```bash
npm install
```

This will install:
- Next.js 15
- React 18
- TypeScript
- All required dependencies

## Step 2: Start Development Server

```bash
npm run dev
```

## Step 3: Open in Browser

Visit: **http://localhost:3000**

---

## That's it! 🎉

Your site should now be running with:
- ✅ All animations working
- ✅ Responsive design
- ✅ Optimized images
- ✅ Fast performance

## Common Commands

```bash
# Development
npm run dev          # Start dev server

# Production
npm run build        # Build for production
npm start            # Start production server

# Code Quality
npm run lint         # Check code quality
```

## Need Help?

1. **Images not loading?** → Check `public/` folder
2. **Port already in use?** → Use `npm run dev -- -p 3001`
3. **Build errors?** → Run `npm install` again

## What's Next?

- Customize content in `src/app/page.tsx`
- Modify styles in `src/app/globals.css`
- Edit components in `src/components/`
- Deploy to Vercel with one click
